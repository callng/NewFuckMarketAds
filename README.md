# NewFuckMarketAds

## 说明 / Desc

NewFuckMarketAds 是一款面向小米应用商店的广告移除 Xposed 模块，通过挂钩应用商店核心逻辑，实现各类广告与冗余功能的屏蔽/移除。

## 功能 / Func

开发中...

- 移除开屏广告
- 隐藏广告低栏
- 隐藏应用安全检测
- 禁止切换前台展示广告
- 将非正版APP重新显示到更新列表
- 显示被商店隐藏的更新
- 移除应用升级页面的软件推荐
- 移除下载页面的软件推荐
- 移除搜索结果的软件推荐
- 移除搜索页面的软件推荐
- 移除搜索建议的软件推荐
- 移除应用详情的评论和推荐
